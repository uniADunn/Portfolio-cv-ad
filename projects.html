Honours Project

IoT-Based Dynamic Light Controller Guided by Historical Solar Irradiance

BSc (Hons) Software Development — Honours Project, Glasgow Caledonian University, 2025–2026

Overview

Indoor hydroponic systems typically use fixed lighting recipes that deliver the right quantity of light but not the natural daily patterns of real sunlight. This project addressed that gap by building and validating an IoT pipeline capable of reproducing historical solar diurnal spectral energy profiles indoors, targeting controlled environment agriculture (CEA) applications.

Stack

Python · Mosquitto MQTT · MySQL · Netbird · AS7341 Spectral Sensor · Raspberry Pi 5 · NASA POWER · ASTM G173 · FAOSTAT · Git / GitHub

Outcome

The engineering hypothesis was supported: the system can reproduce historical diurnal solar spectral energy profiles within defined hardware limits when independent per-band LED control is used and hardware output reaches or exceeds the maximum solar target. The system is ready to progress to biological validation, comparing dynamic solar-guided lighting against static recipes in a live hydroponic grow environment.

Task Management App

Simple Browser-Based Task Manager Built with JavaScript

Self-Study Personal Project, 2026

Overview

Browser-based task managers commonly rely on frameworks and libraries to handle state and DOM updates. This project was built without any, using vanilla JavaScript structured across four focused modules to keep concerns cleanly separated and demonstrate core language fundamentals.

Stack

JavaScript · HTML · CSS · Jest · jsdom · Git / GitHub

Outcome

The app supports creating, updating and deleting tasks with input validation and dynamic re-rendering on every change. A full unit test suite covers all four modules including validation logic, task creation, status updates, deletion and DOM rendering.
